# 📚 Flashcard-o-matic

## 📖 Project Overview

Flashcard-o-matic is a React-based web application that allows users to create, manage, and study flashcard decks. Users can build custom decks, add cards, and test their knowledge through an interactive study mode.

This project demonstrates core front-end development concepts including component-based architecture, state management, routing, and API integration.

---

## ✨ Features

* 🃏 Create, edit, and delete flashcard decks
* 📄 Add, edit, and remove cards within decks
* 🔄 Interactive study mode with card flipping
* 📊 Card progress tracking (e.g., "Card 2 of 5")
* ⚠️ Validation for minimum cards before studying
* 🧭 Navigation using React Router
* 🗂 Breadcrumb navigation for better UX
* 🗑 Confirmation prompts for delete actions

---

## 🛠 Technologies Used

* React
* React Router
* JavaScript (ES6+)
* HTML5 & CSS3
* RESTful API

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/kwingo/Project-_Flashcards-_Qualified_-Node_18--Kirk_Wingo-Solution.git
cd flashcard-app
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm start
```

### 4. Run tests

```bash
npm test
```
